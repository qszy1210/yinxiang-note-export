如何获取 Developer Token

  1. 登录印象笔记网页版
  2. 访问 Developer Token 页面
  3. 点击 "Create a developer token" 按钮
  4. 复制生成的 Developer Token 和 NoteStore URL
注意：请妥善保管您的 Token，不要泄露给他人。

使用本地配置文件

您也可以将 Token 配置在服务器的 .env 文件中，这样每次启动时会自动加载：

  1. 复制 .env.example 为 .env
  2. 编辑 .env 文件，填入您的 Token 和 NoteStore URL
  3. 重新启动服务
EVERNOTE_TOKEN=your_token_here
EVERNOTE_NOTESTORE_URL=https://app.yinxiang.com/shard/s1/notestore

导出格式说明

Obsidian 格式：使用 ![[images/xxx.png]] 语法，适合在 Obsidian 中使用。

标准 Markdown：使用 ![](./images/xxx.png) 语法，兼容大多数 Markdown 编辑器。

常见问题

Token 验证失败？
确保 Token 没有过期或被撤销，确认 NoteStore URL 格式正确。
导出速度慢？
印象笔记 API 有调用频率限制，程序会自动控制请求速度。包含大量图片的笔记需要更多下载时间。
部分内容无法导出？
加密内容无法导出，会显示为提示文字。复杂表格可能格式不完美。
